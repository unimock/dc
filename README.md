# dc  ... docker controller

## Description

Manage generic docker hosts from a central host directory. 

## Installation

```
git clone https://github.com/unimock/dc.git /opt/dc
echo 'export PATH=$PATH:/opt/dc/bin' /etc/profile.d/dc-profile.sh

```
